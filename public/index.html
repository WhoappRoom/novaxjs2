Novaxjs2 About

Novaxjs2 is a lightweight, flexible, and extensible web framework built for Node.js, designed to simplify the process of building and serving dynamic web applications. With a focus on simplicity and performance, Novaxjs2 provides a straightforward API that allows developers to quickly create and manage HTTP servers, handle routing, serve static files, and apply middlewareâ€”all while keeping the code clean and minimal.

Installation

Use the following command to install Novaxjs2:

npm install novaxjs2

Features

  • Route Handling: Define routes using GET, POST, PUT, and DELETE methods.
  • Routes with dynamic parameters (e.g., /user/:id) and (e.g., /search:q ) are matched using regex

  • Middleware Support: Add custom middleware for request handling.

  • Static File Serving: Easily serve static files using the serveStatic method.

  • Customizable Styles: Use a <style> tag to apply custom styles to responses 
    app.style = 'css style'
  • Customizable JavaScript: Use a <script> tag to inject custom JavaScript into responses 
    app.js = '//javascript code'
  • File-based Route Handling: Utilize: Utilize the sendFile method to return file contents directly using 
    app.sendFile(filePath, res).

Usage

Creating the Server

Note: Novaxjs2 version 4.5.7 or above requires initializing the application using 

new nova()

const Nova = require('novaxjs2');

const app = new Nova();

// Define routes
app.get('/', (req, res) => {
    return '<h1>Welcome to Nova Framework!</h1>';
});

// Start the server
app.at(3000, () => {
    console.log('Server is running on port 3000');
});

Middleware


app.useMiddleware((req, res, next) => {
    console.log(`Request Method: ${req.method} - URL: ${req.url}`);
    next();
});

404 Error


app.on(404, (req, res) => {
    return '<h1>404 - Page Not Found</h1>';
});

params access


  app.get('/user/:id', (req, res) => {
    const p = req.params.id;  // Access route parameter 'id'
    return `You searched user with id: ${p}`;
});

query access


  app.get('/search:q', (req, res) => {
    const q = req.query.q;
    return `You searched for: ${q}`;
});

Api reference

get(path, handler): Define a GET route.

post(path, handler): Define a POST route.

put(path, handler): Define a PUT route.
delete(path, handler): Define a DELETE route.

style: Use this to set custom CSS styles.

js: Use this to inject custom JavaScript into your responses.

useMiddleware(middleware): Add middleware to the request handling chain.

on(404, handler): Defines a custom handler for 404 errors. When a requested route is not found, this handler is invoked to return a "Not Found" response or a custom error message.

error(handler): Handle errors with a custom error handler.

sendFile(filePath, res): Send file content directly to the response.

serveStatic(directly): Static files are served using the serveStatic() method, which maps file paths based on the staticPath..

at(port, callback): Start the server and listen on the specified
